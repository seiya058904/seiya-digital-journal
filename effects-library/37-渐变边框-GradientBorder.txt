名称：渐变边框 / GradientBorder
分类：卡片动效
依赖：纯 CSS
使用位置：通用包装器

────────────────────────────────────

【视觉描述】

简单的包装 div，通过 CSS 实现渐变边框效果。

使用 1px padding 技巧配合 background-clip，
在内容周围渲染渐变边框，
同时内部保持正常背景色。

────────────────────────────────────
【技术要点】

- background-clip: padding-box + border-box
- 1px border + gradient background
- 纯 CSS，无 JS 依赖
- 透明度过渡可选
