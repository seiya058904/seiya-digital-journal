名称：边框光晕 / BorderGlow
分类：卡片动效
依赖：纯 CSS + JS 事件
使用位置：Hero 头像、Gallery、Archive 各卡片

────────────────────────────────────

【视觉描述】

卡片的边框和周围区域会发出跟随光标的彩色光晕。

当鼠标靠近卡片时，距离最近的边缘亮起，
产生多层 inset/outset box-shadow 光晕，
颜色可自定义（紫色、粉色、蓝色等）。

锥形遮罩（conic mask）将光晕限制在
从光标角度发出的锥形区域内，
实现「光从你手指方向打过来」的感觉。

此外有 7 个径向渐变色块叠加在边框上，
形成微妙的彩虹边缘效果。

animated 模式下，卡片首次出现时会播放
一次扫光动画——光晕从一侧亮起、
沿边框扫一圈、然后淡出。

移动端/触摸设备禁用。

────────────────────────────────────
【技术要点】

- CSS conic-gradient mask 实现方向性光晕
- mousemove 事件计算光标到各边缘距离
- 多层 box-shadow 叠加产生光晕深度
- animated 模式使用 CSS animation 一次性播放
- 移动端通过 hover 媒体查询检测并禁用
