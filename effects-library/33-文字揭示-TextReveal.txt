名称：文字揭示 / TextReveal
分类：滚动动效
依赖：Framer Motion
使用位置：页面标题区

────────────────────────────────────

【视觉描述】

文字从容器下方 108% 处（被 CSS overflow clip 隐藏），
滑升到自然位置，持续 0.82 秒。

使用与 ScrollReveal 相同的自定义缓动。

产生"从地面升起"的效果——
只有文字的可见部分被父元素的 overflow 裁剪，
看起来文字正从遮罩中被揭开。

组件挂载时立即触发（非滚动触发）。

────────────────────────────────────
【技术要点】

- CSS overflow: clip 实现文字裁剪遮罩
- Framer Motion 驱动 translateY
- initial: { y: '108%' } → animate: { y: '0%' }
- duration: 0.82s
- ease: [0.22, 1, 0.36, 1]
- 挂载即触发，不依赖滚动
