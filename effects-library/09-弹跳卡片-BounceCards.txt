名称：弹跳卡片 / BounceCards
分类：卡片动效
依赖：GSAP
使用位置：Motion Lab

────────────────────────────────────

【视觉描述】

一排图片卡片以弹性弹跳方式入场——
从 scale 0 弹到 scale 1，逐张交错出现。

每张卡片有预设的旋转角度和水平偏移，
从中心向外扇形展开。

hover 时，被悬停的卡片回正（消除旋转），
其他卡片根据相对位置向左或向右滑开，
形成展开/散开效果。

使用 GSAP elastic 缓动，有明显的弹性回弹感。

────────────────────────────────────
【技术要点】

- GSAP elastic easeOut 实现弹跳入场
- stagger 控制逐张延迟
- hover 时 GSAP 动画其他卡片位移
- 每张卡片预设 transform: rotate + translateX
