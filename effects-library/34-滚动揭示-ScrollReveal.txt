名称：滚动揭示 / ScrollReveal
分类：滚动动效
依赖：Framer Motion
使用位置：全站所有区块（About、Interests、Gallery、Thoughts、Journey、Contact、Archive 各页面）

────────────────────────────────────

【视觉描述】

Framer Motion 包装器，子内容从下方 34px 处
和完全透明状态，滚动到视口时滑入到自然位置和透明度。

使用自定义 cubic-bezier 缓动（0.22, 1, 0.36, 1），
持续 0.8 秒。

动画在元素每次滚回视口时重播（once: false）。

可配置延迟和视口交叉阈值。

这是全站使用最广泛的动效包装器，
几乎所有区块内容都通过它入场。

────────────────────────────────────
【技术要点】

- Framer Motion motion.div
- initial: { opacity: 0, y: 34 }
- animate: { opacity: 1, y: 0 }
- viewport: { once: false } 每次重播
- 自定义 ease: [0.22, 1, 0.36, 1]
- duration: 0.8s
