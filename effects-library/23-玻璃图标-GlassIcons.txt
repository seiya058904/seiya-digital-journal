名称：玻璃图标 / GlassIcons
分类：卡片动效
依赖：纯 CSS + JS 事件
使用位置：Notes Vault 分类导航

────────────────────────────────────

【视觉描述】

图标磁贴网格。每块磁贴有三层：
- 彩色背面板（旋转 15°）
- 磨砂玻璃前面板（模糊 + 白色内边框）
- 居中的图标

hover 时：
- 背面板进一步旋转并斜向位移
- 前面板抬升（translateZ）
- 标签文字在下方淡入并上浮

玻璃磁贴有 3D 视差分离效果。

────────────────────────────────────
【技术要点】

- CSS 3D transform 分层实现视差
- backdrop-filter: blur() 实现磨砂玻璃
- hover 时 rotate / translateZ 动画
- 标签淡入 + translateY 上浮
- 纯 CSS 动画，无 JS 依赖
