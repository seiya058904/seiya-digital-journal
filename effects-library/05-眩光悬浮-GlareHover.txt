名称：眩光悬浮 / GlareHover
分类：卡片动效
依赖：纯 CSS
使用位置：Gallery 卡片

────────────────────────────────────

【视觉描述】

深色矩形容器表面，hover 时一条斜向白色眩光条
从左上角滑向右下角（角度约 -45°），
持续约 650ms。

眩光是一条窄的渐变带（透明→白色→透明），
像一道反光从卡片表面划过。

播放一次后重置（不可见状态），
下次 hover 再次触发。

触摸/移动设备上完全隐藏眩光层。

────────────────────────────────────
【技术要点】

- 纯 CSS animation + transform 实现
- 无 JS 依赖
- hover 触发，播放一次后停止
- touch 设备通过 media query 隐藏
