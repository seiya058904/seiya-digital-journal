名称：文件夹 / Folder
分类：卡片动效
依赖：纯 CSS + JS 事件
使用位置：Archive 首页 — 区块导航

────────────────────────────────────

【视觉描述】

3D CSS 文件夹插图，点击开合。

关闭状态：显示彩色文件夹正面，
边缘微微露出纸张边角。

点击后：文件夹盖板打开，
最多 3 张白色"纸张"向上扇形展开。

每张纸可以通过光标磁力微调位置
（小幅度视差偏移跟随鼠标）。

文件夹和纸张使用 CSS 自定义属性控制配色。

────────────────────────────────────
【技术要点】

- CSS 3D transform 实现文件夹开合
- click 事件切换 open/closed 状态
- 纸张使用 Framer Motion 或 CSS 动画展开
- mousemove 计算纸张视差偏移
- CSS 自定义属性控制颜色主题
