名称：全息名片 / ProfileCard
分类：卡片动效
依赖：纯 CSS + requestAnimationFrame
使用位置：Hero 头像区、Motion Lab

────────────────────────────────────

【视觉描述】

3D 透视名片，跟随光标产生倾斜效果。

卡片有多层视觉叠加：
- behind-glow：卡片背后透出的柔和彩色光
- 内部渐变填充层
- 光泽层（shine）
- 眩光层（glare）

所有层通过 CSS 自定义属性驱动，
随倾斜角度实时更新。

首次加载时，倾斜动画从右上角开始，
缓动回中心位置。

卡片内含头像图片、迷你头像（带用户名/状态），
以及一个联系按钮。

整个倾斜引擎使用 requestAnimationFrame 保持 60fps 流畅。

────────────────────────────────────
【技术要点】

- CSS perspective + rotateX/Y 实现 3D 倾斜
- RAF 循环读取光标位置并更新 CSS 变量
- 多层叠加：glow / gradient / shine / glare
- 弹簧插值平滑光标跟随
- 首帧入场动画从偏置位置回中
