## 5. 建议如何选择

### 选择规则

#### art picture — 选 8-10 张

| 规则 | 说明 |
|---|---|
| 优先深色 / 冷色调 | 符合 dark editorial 风格 |
| 优先抽象 / 几何 / 光影 | 不要人物、卡通、搞怪 |
| 优先竖图（736×1308 类型） | 适合 Gallery portrait 卡片 |
| 选 1-2 张方图 | 适合 square 卡片，增加节奏感 |
| 选 1-2 张超长竖图 | 适合 featured portrait 大卡 |
| 排除低分辨率（< 500px） | #14 已排除 |
| 排除人物 / 品牌 / 角色 | #1 #40 #42 已排除 |

#### picture — 选 10-20 张，首批入 8-12 张

| 规则 | 说明 |
|---|---|
| 优先横图（landscape） | 适合 wide / large 卡片展示 |
| 竖图用于 portrait 卡片 | 构图要好，不要随手拍 |
| 避免纯自拍、太私人的 | 优先远景、侧脸、背影、剪影 |
| 避免他人正脸明显 | 隐私考虑 |
| 优先有故事感的 | 旅行、学习、夜晚、雨天、城市 |
| 相似场景只选最佳 1 张 | 连拍照片只留一张 |
| 生活照总量不超过 6 张 | 第一批先选 2-3 张 |
| 学习/工作场景照选 2-3 张 | 适合 Learning section |

### 如何配合 contact sheet 使用

1. 打开 `report/contact-sheet-art-picture.jpg`
2. 浏览所有缩略图，记下你喜欢的编号（如 #4, #8, #15...）
3. 打开 `report/contact-sheet-picture-01.jpg` 和 `02.jpg`
4. 同样记下你认可的生活照编号
5. 告诉我编号，我来做：WebP 压缩 → 命名 → 接入
